# 🧾 Java Billing System

A desktop-based billing system built using **Java Swing**.  
This project allows users to generate and manage customer bills through a clean, intuitive GUI. Ideal for small retail environments or academic demonstrations.

---

## 🚀 Features

- 🛒 Add items with price and quantity
- 💰 Auto calculate totals, tax, and discounts
- 🖨️ Print and save generated bills
- 📋 Basic customer info handling
- 🧑‍💻 Built with Java Swing (GUI)
